🍽 Simply Savory API

Welcome to the Simply Savory recipe data server. Use the endpoints below to access recipe information.

Available Endpoints

GET

/api/recipes

Returns all recipes with full details including ingredients, instructions, ratings, and images.

GET

/api/recipes/:id

Returns a single recipe by its ID. Example: /api/recipes/1 returns Creamy Garlic Pasta.
